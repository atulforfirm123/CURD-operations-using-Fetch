<!-- This repository contains a simple ToDo List application built using HTML, CSS, and JavaScript. The app utilizes the JSONPlaceholder API (https://jsonplaceholder.typicode.com/) to perform CRUD (Create, Read, Update, Delete) operations on posts.

Features
GET Posts: Retrieves a list of posts from the JSONPlaceholder API and displays them in a table format.
CREATE Post: Creates a new post and adds it to the list of posts.
UPDATE Post: Updates an existing post with new content.
DELETE Post: Deletes a post from the list.

Technologies Used
HTML: Markup language used for structuring the web page.
CSS (Bootstrap): Provides styling and layout for the web page elements.
JavaScript: Programming language used for interactive features and API interaction.
JSONPlaceholder API: Provides a mock REST API for testing and prototyping.

Usage
Clone the repository to your local machine.
Open the index.html file in a web browser.
Use the buttons provided to interact with the ToDo List application:
Click "GET" to retrieve and display existing posts.
Click "CREATE" to add a new post.
Click "PUT" to update an existing post.
Click "DELETE" to remove a post.

Files
index.html: Main HTML file containing the structure of the web page and buttons for CRUD operations.
script.js: JavaScript file containing the logic for interacting with the JSONPlaceholder API and handling user actions.
style.css: External CSS file providing additional styling, including Bootstrap for responsive design.
Dependencies
Bootstrap: Used for styling and responsive layout. The CDN link is included in the HTML file. -->
